# Silver Bullion P2P 贷款市场监控

实时监控 [Silver Bullion](https://www.silverbullion.com.sg) P2P 贷款市场的利率趋势、成交量及基准利率对比，帮助出借人制定更合理的挂单策略。

**在线访问：** https://amoukyou.github.io/silver-monitor/

---

## 关于 Silver Bullion

Silver Bullion 是一家总部位于新加坡的贵金属存储与 P2P 借贷平台，成立于 2009 年。平台是**伦敦金银市场协会（LBMA）的认可合作伙伴**，旗下的 S.T.A.R. Storage 系统由独立第三方审计，托管资产定期公开核查。

---

## 抵押贷款运作原理

Silver Bullion 的 P2P 贷款是一种**以实物贵金属为抵押的点对点借贷**，流程如下：

1. **借款人抵押贵金属**：借款人将黄金或白银存入 Silver Bullion 的托管金库，平台按当前市价评估抵押品价值
2. **发起借款订单**：借款人设定借款金额（SGD/USD/EUR）、期限（1/6/12/24 个月）和愿意支付的年利率
3. **出借人撮合成交**：出借人在市场中选择合适的订单，资金直接划转给借款人
4. **到期还款**：借款人按约定利率还本付息，出借人收回资金
5. **抵押品保护**：若抵押品市值跌破警戒线，平台将触发追加保证金或**强制变卖抵押品**优先偿还出借人，以此保护出借资金安全
6. **超额抵押**：平台要求抵押品价值通常高于借款金额的 200%（即贷款价值比 LTV ≤ 50%），为出借人提供较厚的安全垫

## 功能一览

### 数据图表

- **利率趋势图** — 按币种（SGD/USD/EUR）和期限（1M/6M/12M/24M）查看平均净年化利率走势，支持净/毛利率切换及利差区间可视化
- **成交量趋势** — 每周成交规模变化，直观反映市场活跃度
- **利率散点图** — 每笔成交的净年化利率分布，观察利率离散程度
- **成交规模散点图** — 每笔成交金额分布，附每日中位数趋势线

### 工具面板

- **近期成交** — 按当前币种和期限浏览历史成交记录，含毛利率/净年化、金额、分页导航
- **利率估算** — 输入期限和金额，根据近30天成交分位数推算竞争力利率及预期净收益
- **订单回顾** — 输入已有订单信息，计算持有至今和到期的净利息
- **基准对比** — 自动获取美国国债（USD）、ECB 存款利率（EUR）、HSBC 新加坡定存（SGD），显示 P2P 与银行存款的利差

### 交互体验

- **全局币种联动** — 右侧图表的币种按钮同时控制左侧工具面板，一键切换
- **数据间断提示** — 自动检测超过 7 天的交易数据缺口并显示黄色提示
- **异步数据加载** — 页面骨架即时渲染，数据后台加载，附加载动画和错误处理
- **无障碍** — 利差涨跌用 ▲/▼ 符号辅助，不仅依赖红/绿颜色
- **响应式** — 适配桌面、平板（768px）及手机端

## 数据说明

- 数据来源：Silver Bullion 官网公开成交报告（PDF）
- 每 4 小时自动抓取更新一次，页面显示具体数据范围（如 2025-09-05 至 2026-03-05）
- 管理费扣除规则：1 个月期 **1.00% p.a.**，其余期限 **0.50% p.a.**

## 项目结构

```
index.html          静态前端页面（通过 fetch 加载数据）
fetch-data.js       数据抓取脚本（PDF 解析 + 基准利率获取）
data.json           交易数据（自动更新）
benchmark.json      基准利率数据（自动更新）
.github/workflows/  GitHub Actions 定时任务
```

---

## 技术栈

- **数据抓取**：Node.js + pdf-parse → 输出 `data.json` + `benchmark.json`
- **前端**：静态 HTML + Chart.js 4.x + chartjs-adapter-date-fns，通过 `fetch()` 异步加载数据
- **自动部署**：GitHub Actions（每 4 小时定时任务）+ GitHub Pages（静态托管）

## 风险提示

> 本项目仅作数据展示与辅助分析，**不构成任何投资建议**。出借资金前请充分了解以下风险：

- **贵金属价格风险**：抵押品（黄金/白银）价格剧烈下跌时，即使触发强制平仓，极端行情下仍可能造成出借人损失
- **平台风险**：P2P 平台存在运营、监管、流动性等风险，Silver Bullion 并非银行，出借资金不受存款保险保障
- **流动性风险**：贷款期限内资金被锁定，无法提前赎回；若市场挂单量少，可能出现难以成交的情况
- **汇率风险**：以外币（USD/EUR）出借时，须承担汇率波动带来的本金折算损失
- **利率风险**：市场利率受宏观经济影响波动，历史利率不代表未来收益
